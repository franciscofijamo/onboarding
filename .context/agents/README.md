# Agent Playbooks

Use these short playbooks to bootstrap work in this repo.

## Available Guides
- `architecture-planning.md` for scoping changes and tracing dependencies.
- `frontend-development.md` for UI, routing, and component work.
- `backend-development.md` for API, domain logic, and integrations.
- `database-development.md` for Prisma and schema work.
- `security-check.md` for auth, env, and data-handling review passes.

## Working Rule
- Choose the closest playbook, then pair it with the topic docs in `.context/docs/`.
